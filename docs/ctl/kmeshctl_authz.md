## kmeshctl authz

Enable or disable Kmesh's authz

```
kmeshctl authz [flags]
```

### Examples

```
# Enable Kmesh's authz:
 kmeshctl authz <kmesh-daemon-pod> enable
 
 # Disable Kmesh's authz:
 kmeshctl authz <kmesh-daemon-pod> authz
```

### Options

```
  -h, --help   help for authz
```
